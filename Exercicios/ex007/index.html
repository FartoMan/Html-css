Semântica

A HTML5 utiliza muito da semântica, ou seja, ele dá mais valor ao significado. Na HTML4 as tags eram mais voltadas para a forma como quesito visual, já a HTML5 utiliza a folha de estilo CSS para este trabalho, utilizando as tags para dar o valor semântico, que é voltado à linguagem em si para ser trabalhado dentro dela (na folha de estilo) a forma.

Mas como isso pode dar mais valor à linguagem?

Utilizando esses parâmetros, a HTML5 junto com CSS3 e, posteriormente, com o JavaScript, trabalham em multiplataformas entregando mais interação e adaptação a cada plataforma, de acordo com o funcionamento de cada uma.
