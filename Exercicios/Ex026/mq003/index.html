Mudando a orientação do dispositivo, o pode ser reposicionado e apresentado de formas diferentes, cada um tendo as configurações feitas para cada tipo de experiência do usuário.

Cada dispositivo é experiênciado de uma forma diferente — o celular com a ponta dos dedos; o computador com teclado e mouse; o tablete, que é maior que um celular, pode estar sendo utilizado até mesmo com uma caneta específica.
