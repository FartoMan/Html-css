Meu primeiro exercício com Media Query

Media Queries (MQ) são formas de exibir o conteúdo em diferentes tipos de telas com configurações diferentes. Ao invés de criar um arquivo personalizado para cada configuração, criamos configurações para cada formato sem ter que alterar o seu conteúdo.

Nesse exercício, por exemplo, praticamos a personalização do conteúdo na tela do computador e para o formato para impressão (Crtl + P). Quando comparado, vemos claramente que cada conteúdo é disposto e exibido de uma forma diferente nesses dois formatos.

Usando esse exemplo de formato tela do computador e impresso, vemos a diferença do que realmente é necessário ser exibido ou não, como o menu de navagação e o espaçamento, como também o tipo de fonte utilizado além do texto ao final do artigo.

No arquivo html, utilizamos dois links para acessar duas folhas de estilo diferentes sem afetar o estilo de uma ou de outra quando forem ser exibidos a partir de cada media-type.

O que é Media Query?

Media Query é a união de Media Type e Media Feature... Como assim? Bom, Media Type é o tipo de mídia, já Media Feature são as características da mídia
