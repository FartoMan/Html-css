Atributos

Neste formulário, é solicitado o preenchimento da senha,
por isso utilizamos o método POST.
Também vemos os atributos:
required -> para o envio do formulário, este campo tem que ser preenchido
minlength -> quantidade mínima de caracteres para ser válidado
maxlength -> quantidade máxima de caracteres para ser validado
size -> tamanho fixo do campo em caracteres (size="10")
placeholder -> Texto 'explicativo' dentro do campo
