Atributos

action -> Onde serão processados os dados enviados
method -> get/post é como os dados serão processados
get -> exibe o resultado na URL; máximo de 3000bits
post -> melhor para senhas e arquivos
autocomplete on/off -> Ao clicar no campo, exibe as sugestões de completar com os dados já salvos no navegador do usuário
