Inserindo imagens flexíveis


        <picture>
            <source media="(max-width: 720px)"
            srcset="Imagens/panpit-cel.jpg" type="image/jpg">
            <img src="Imagens/panpit-larg.JPG"
            alt="Fotografia de duas cadelas deitadas dormindo,
            uma no sofá e outra logo abaixo do sofá">
        </picture>

Imagens "flexíveis" é um recurso da HTML que altera uma imagem quando o navegador apresentar diferentes dimensões (em píxels). Ou seja, podemos alterar uma imagem para uma escala menor para se adequar às dimensões de cada navegador, evitando assim a barra de rolagem para a sua visualização. O ideal é que seja para se adequar melhor a diversos dispositivos.

  • É interessante criar alguns arquivos em escala reduzida para melhor desempenho do site em diversos dispositivos. O arquivo é carregado pelo atributo srcset na tag <source>
  • A tag <source> deve vir sempre antes da tag <img>, pois o navegador lerá primeiro como ele irá se adequar para a apresentação da imagem, lendo as especificações dadas no atributo media="(max-width ou min-width: 000px)".
  • Ideal para reduzir o uso de dados de rede em dispositivos móveis.
  • Para mais informações, acesse W3School: HTML picture tag.
