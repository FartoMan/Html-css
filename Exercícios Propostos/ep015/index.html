Seletores CSS id e class

Seletor ID

Como usar

Para utilizar um seletor id, aplicamos ele dentro da tag HTML e acionamos id="nome".

Temos que aplicar dentro da folha de estilos (ou na área <style> no cabeçalho) um símbolo de # mais o nome, dessa forma:

        
            #nome {
                color: #FF0000
            }

Como funciona

Ao aplicar um id a um elemento HTML, esse elemento aplicará as atribuições especificadas a ele, junto com as dadas as personalizações dadas diretamente à tag.

Temos a seguir seguintes atribuições à tag <p>:

        
            p {
                font-size: 16px;
            }

            #grande {
                font-size: 20px;
                color: green;
            }

Seletor Class

Como usar

Semelhante ao id, dentro da tag, utilizamos class="titulo".

Devendo estar especificado dentro da CSS com um . + o nome dado, no caso, titulo, dessa maneira:

Header

        
            .titulo {
                text-weight: bold;
            }

Body

        
            <h1 class="titulo">Título 1<h1>

Como funciona

Idêntico ao id, a aplicação unirá às atribuições já dadas à tag, ou irá sobrepor uma mesma atribuição que tenha valores diferentes.

Podemos utilizar o mesmo exemplo anterior:

        
            p {
                font-size: 16px;
            }

            .grande {
                font-size: 20px;
                color: green;
            }
